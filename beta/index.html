Cats Blender Plugin

This website is maintained by The Neoneko team, who maintain the Unofficial Cats Plugin. Our aim is to make it easier for you to choose the correct version of Cats for your Blender version and to get help with your specific version of Cats.

Select the Cats Blender Plugin version that best suits your needs. Please carefully read the descriptions for each version and ensure you use the appropriate support channels if you require assistance.

Official Cats

The Official Cats plugin supports Blender versions up to 3.6. However, it may have some issues on Blender 3.6 due to outdated MMD Tools. We recommend using the Official Cats plugin on Blender 3.5 and below.

Unofficial Cats

The Unofficial Cats plugin supports Blender versions 3.6 to 4.2. It features the latest MMD Tools and includes numerous fixes and improvements. This version is recommended for users of Blender 3.6 and above.
